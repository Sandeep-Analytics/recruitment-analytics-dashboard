# recruitment-analytics-dashboard

## 📊 Overview

This project focuses on analyzing recruitment data and building an interactive dashboard to track hiring performance.

## 🛠 Tools & Technologies

* Power BI
* Excel
* Power Query (ETL)
* DAX

## 📈 Key Metrics

* Submissions, Interviews, Offers, Joins
* Conversion Ratio
* Time to Fill
* Recruiter Performance
* Pipeline Aging

## 🔍 Key Insights

* Identified bottlenecks in the hiring pipeline
* Improved visibility into recruitment performance
* Helped optimize recruiter efficiency
